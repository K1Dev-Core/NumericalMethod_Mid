### **ส่วนที่ 1: พื้นฐานวิธีเชิงตัวเลขและแบบจำลอง**

**1. ตามกระบวนการแก้ปัญหา ขั้นตอนใดที่อยู่ระหว่าง "ปัญหา" และ "ผลลัพธ์"?**
*   ก. การคำนวณด้วยแรงคน
*   **ข. แบบจำลองทางคณิตศาสตร์ (Mathematical Model)**
*   ค. การสุ่มเดาคำตอบ
*   ง. การเขียนรายงาน
*   **เฉลย:** ข. แบบจำลองทางคณิตศาสตร์ คือสิ่งที่ใช้เปลี่ยนปัญหาให้เป็นผลลัพธ์โดยอาศัยกฎและสมการต่าง ๆ

**2. ในการสร้างแบบจำลองทางคณิตศาสตร์ เราสามารถนำสิ่งใดมาใช้ได้บ้าง?**
*   ก. กฎทางฟิสิกส์
*   ข. กฎทางคณิตศาสตร์
*   ค. สมการทางคณิตศาสตร์
*   **ง. ถูกทุกข้อ**
*   **เฉลย:** ง. การสร้างแบบจำลองต้องใช้ทั้งกฎทางฟิสิกส์ คณิตศาสตร์ และสมการที่เกี่ยวข้องประกอบกัน

**3. "วิธีเชิงวิเคราะห์" (Analytical Methods) มีเป้าหมายเพื่อให้ได้ผลลัพธ์แบบใด?**
*   **ก. ผลเฉลยเชิงวิเคราะห์ (Exact Value)**
*   ข. ผลเฉลยเชิงตัวเลข (Numerical Value)
*   ค. ค่าที่เกิดจากความคลาดเคลื่อน
*   ง. ค่าจากการประมาณการ
*   **เฉลย:** ก. วิธีเชิงวิเคราะห์จะให้คำตอบที่ถูกต้องแม่นยำหรือ Exact Value

**4. "วิธีเชิงตัวเลข" (Numerical Methods) แตกต่างจากวิธีเชิงวิเคราะห์อย่างไรในด้านคำตอบ?**
*   ก. ให้คำตอบที่ถูกต้องที่สุดเสมอ
*   **ข. ให้ผลเฉลยเชิงตัวเลขที่เป็นค่าโดยประมาณและมักมีความคลาดเคลื่อนเกิดขึ้น**
*   ค. ไม่ต้องใช้คอมพิวเตอร์ในการคำนวณ
*   ง. ให้คำตอบเป็นสมการติดตัวแปรเสมอ
*   **เฉลย:** ข. วิธีเชิงตัวเลขให้ Numerical Value ซึ่งเป็นค่าโดยประมาณและมี Error เกิดขึ้น

**5. จากตัวอย่างการแก้สมการ $x^2 - 2\sqrt{2}(x) + 2 = 0$ ข้อใดคือคำตอบแบบ Exact Value?**

*   **เฉลย:** ค. ผลเฉลยเชิงวิเคราะห์ของสมการนี้คือ $x = \sqrt{2}$

### **ส่วนที่ 2: ค่าความคลาดเคลื่อนและการคำนวณ**

**6. หาก $x$ คือค่าจริง และ $\tilde{x}$ คือค่าประมาณ สูตรความคลาดเคลื่อน ($e$) คือข้อใด?**
*   **ก. $e = x - \tilde{x}$**
*   ข. $e = \tilde{x} - x$
*   ค. $e = |x - \tilde{x}|$
*   ง. $e = x / \tilde{x}$
*   **เฉลย:** ก. Error (e) เท่ากับ Exact value ลบด้วย Approximate value

**7. ความคลาดเคลื่อนสัมบูรณ์ (Absolute Error, $e_{abs}$) มีสูตรว่าอย่างไร?**
*   ก. $e_{abs} = x - \tilde{x}$
*   **ข. $e_{abs} = |x - \tilde{x}|$**
*   ค. $e_{abs} = |x| - |\tilde{x}|$
*   ง. $e_{abs} = (x - \tilde{x})^2$
*   **เฉลย:** ข. ความคลาดเคลื่อนสัมบูรณ์คือค่าสัมบูรณ์ของผลต่างระหว่างค่าจริงและค่าประมาณ

**8. สูตรของความคลาดเคลื่อนสัมพัทธ์ (Relative Error, $e_{rel}$) คือข้อใด?**
*   ก. $e_{rel} = |x - \tilde{x}|$
*   **ข. $e_{rel} = |x - \tilde{x}| / |x|$**
*   ค. $e_{rel} = |x| / |x - \tilde{x}|$
*   ง. $e_{rel} = (x - \tilde{x}) \times 100$
*   **เฉลย:** ข. ความคลาดเคลื่อนสัมพัทธ์หาได้จากความคลาดเคลื่อนสัมบูรณ์หารด้วยค่าจริง

**9. ข้อใดจัดเป็นที่มาของความคลาดเคลื่อน (Source of Error) ตามแหล่งข้อมูล?**
*   ก. จากโมเดลทางคณิตศาสตร์ และจากการปัดเศษ
*   ข. จากการตัดปลาย และจากข้อมูลนำเข้า
*   ค. จากเครื่องคำนวณ และจากการปฏิบัติ
*   **ง. ถูกทุกข้อ**
*   **เฉลย:** ง. แหล่งข้อมูลระบุที่มาของ Error ไว้ 6 แหล่งตามที่กล่าวมาทั้งหมด

**10. "Propagation Error" มีนิยามอย่างไร?**
*   ก. ความคลาดเคลื่อนที่เกิดจากการปัดเศษทศนิยม
*   **ข. ความคลาดเคลื่อนจากข้อมูลนำเข้าที่ส่งผลต่อเนื่องไปยังผลลัพธ์ผ่านกระบวนการคำนวณ**
*   ค. ความคลาดเคลื่อนที่เกิดจากการตัดพจน์ในอนุกรม
*   ง. ความคลาดเคลื่อนจากการใช้กฎทางฟิสิกส์ผิด
*   **เฉลย:** ข. คือการที่ข้อมูลที่มี Error อยู่แล้วถูกนำเข้าสู่กระบวนการและส่งผลต่อผลลัพธ์สุดท้าย

**11. ความคลาดเคลื่อนจากการตัดปลาย (Truncation error) เกิดขึ้นจากอะไร?**
*   ก. ข้อมูลนำเข้าผิดพลาด
*   **ข. การประมาณโดยใช้สูตรแล้วมีการตัดพจน์ทิ้ง**
*   ค. การปัดเศษทศนิยมในคอมพิวเตอร์
*   ง. การพิมพ์สูตรผิด
*   **เฉลย:** ข. เกิดจากการใช้ระเบียบวิธีประมาณค่าแล้วตัดพจน์บางพจน์ทิ้งเพื่อให้คำนวณได้

**12. การใช้ "อนุกรมแมคคลอริน" เพื่อประมาณค่าฟังก์ชัน จัดเป็นความคลาดเคลื่อนประเภทใด?**
*   ก. Round-off error
*   **ข. Truncation error**
*   ค. Human error
*   ง. Model error
*   **เฉลย:** ข. การใช้อนุกรมแมคคลอรินเป็นการประมาณค่าที่ต้องตัดพจน์ที่เหลือทิ้ง จึงเกิด Truncation error

**13. การคำนวณใน Python เช่น `math.sqrt(3) * math.sqrt(3)` แล้วได้ `2.9999999999999996` เป็นตัวอย่างของอะไร?**
*   **ก. Round-off error**
*   ข. Truncation error
*   ค. Input data error
*   ง. Mathematical model error
*   **เฉลย:** ก. เกิดจากการที่คอมพิวเตอร์เก็บค่าทศนิยมได้จำกัดจึงต้องมีการปัดเศษ (Round-off)

**14. Computational error จะเกิดขึ้นเมื่อใดตามแหล่งข้อมูล?**
*   ก. เมื่อเขียนโปรแกรมไม่เสร็จ
*   **ข. เมื่อนำค่าที่เกิดจากการปัดเศษไปคำนวณต่อ ทำให้เกิดการสะสมความคลาดเคลื่อน**
*   ค. เมื่อใช้เครื่องคิดเลขที่แบตเตอรี่ต่ำ
*   ง. เมื่อไม่มีข้อมูลนำเข้า
*   **เฉลย:** ข. การนำค่าที่มีการปัดเศษไปใช้คำนวณในขั้นตอนถัดไปจะทำให้ Error สะสมและส่งต่อกันไป

**15. จากตัวอย่างการประมาณ $x$ และ $y$ โดยปัดเศษเหลือ 3 ตำแหน่งเพื่อหา $x - y$ ทำให้เกิดความคลาดเคลื่อนสัมพัทธ์ประมาณเท่าใด?**
*   ก. 0
*   ข. 0.001
*   **ค. 0.037691**
*   ง. 0.5
*   **เฉลย:** ค. จากการคำนวณตัวอย่าง ความคลาดเคลื่อนสัมพัทธ์ที่ได้คือ 0.037691

### **ส่วนที่ 3: ทบทวนคณิตศาสตร์**

**16. ค่าอนุพันธ์ $f'(x_0)$ มีความหมายเชิงเรขาคณิตคืออะไร?**
*   ก. พื้นที่ใต้กราฟ
*   **ข. ความชันของเส้นสัมผัสกราฟที่จุด $x_0$**
*   ค. จุดต่ำสุดของฟังก์ชัน
*   ง. ระยะทางเฉลี่ย
*   **เฉลย:** ข. อนุพันธ์ ณ จุดใดๆ คือความชันของเส้นตรงที่มาสัมผัสกราฟ ณ จุดนั้น

**17. การหาปริพันธ์ (Integration) เป็นการดำเนินงานที่คู่กับการดำเนินงานใด?**

*   **ข. การหาอนุพันธ์ (Differentiation)**
*   ค. การยกกำลัง
*   ง. การถอดราก
*   **เฉลย:** ข. การหาปริพันธ์คือกระบวนการตรงข้ามกับการหาอนุพันธ์

**18. ปริพันธ์จำกัดเขตของ $f(x)$ บนช่วง $[a, b]$ หาได้จากสูตรใด?**

**19. สมการเส้นตรงรูปแบบ $y = mx + c$ ตัวแปร $m$ หมายถึงอะไร?**

**20. สูตรการหาความชัน ($m$) จากจุดสองจุด $(x_1, y_1)$ และ $(x_2, y_2)$ คือข้อใด?**

*   ง. $m = (y_1 - y_2) / (x_2 - x_1)$

**21. เราสามารถหาความชันของเส้นตรงได้จากฟังก์ชันตรีโกณมิติใด?**

**22. รากของสมการ $ax^2 + bx + c = 0$ หาได้จากสูตรใด?**
