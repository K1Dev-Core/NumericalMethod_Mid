## การรันโปรแกรม

โปรเจคนี้มี 5 โปรแกรมสำหรับคำนวณวิธีเชิงตัวเลข:

### 1. Bisection Method (วิธีครึ่งช่วง)
```bash
python bisection_generator.py
```
- สร้างไฟล์: `Bisection_Result_Python.xlsx`
- สมการ: x^3 + 3*x^2 - 1
- ช่วง: [0, 1]

### 2. False Position Method (วิธีตำแหน่งเท็จ)
```bash
python false_position.py
```
- สร้างไฟล์: `False_Position_Result.xlsx`
- สมการ: x^4 + 2*x^2 - x - 3
- ช่วง: [-1.5, 0.5]

### 3. Newton-Raphson Method (วิธีนิวตัน-ราฟสัน)
```bash
python newton.py
```
- สร้างไฟล์: `Newton_Result.xlsx`
- สมการ: x^4 + 2*x^2 - x - 3
- ค่าเริ่มต้น: x0 = 1
- คำนวณอนุพันธ์อัตโนมัติด้วย sympy

### 4. Secant Method (วิธีเส้นสัมผัส)
```bash
python secant.py
```
- สร้างไฟล์: `Secant_Result.xlsx`
- สมการ: sin(x) - x^2
- ค่าเริ่มต้น: x0 = 0.75, x1 = 1.0

### 5. Cramer's Rule (กฎของเครเมอร์)
```bash
python cramers_rule.py
```
- สร้างไฟล์: `Cramer_Rule_Result.xlsx`
- แก้ระบบสมการเชิงเส้น: AX = B
- เมทริกซ์ 3x3

## คำอธิบายวิธีเชิงตัวเลข

### 1. Bisection Method (วิธีครึ่งช่วง)
- **หลักการ**: แบ่งช่วง [a,b] ที่มีรากอยู่ภายในออกเป็นสองส่วนเท่าๆ กันเรื่อยๆ จนกว่าความกว้างช่วงจะน้อยกว่าค่าความคลาดเคลื่อนที่กำหนด
- **ข้อดี**: หาค่าได้แน่นอนเสมอ (converge)
- **ข้อเสีย**: ใช้จำนวนรอบมาก
- **เหมาะสำหรับ**: หาค่าเริ่มต้นของรากเมื่อไม่ทราบค่าโดยประมาณ

### 2. False Position Method (วิธีตำแหน่งเท็จ)
- **หลักการ**: คล้ายวิธีครึ่งช่วง แต่ใช้การสร้างเส้นตรงเชื่อมจุด (a,f(a)) และ (b,f(b)) แล้วหาจุดตัดแกน x เป็นค่าประมาณใหม่
- **ข้อดี**: เร็วกว่าวิธีครึ่งช่วงมาก
- **ข้อเสีย**: อาจ converge ช้าในบางกรณี
- **เหมาะสำหรับ**: ฟังก์ชันที่เรียบๆ ในช่วงที่สนใจ

### 3. Newton-Raphson Method (วิธีนิวตัน-ราฟสัน)
- **หลักการ**: ใช้ค่าอนุพันธ์ของฟังก์ชันในการหาค่าประมาณถัดไป โดยใช้สูตร: x_{n+1} = x_n - f(x_n)/f'(x_n)
- **ข้อดี**: เร็วมาก (quadratic convergence)
- **ข้อเสีย**: ต้องการค่าอนุพันธ์ และอาจ diverge ถ้าค่าเริ่มต้นไม่เหมาะสม
- **เหมาะสำหรับ**: กรณีที่ทราบค่าเริ่มต้นที่ใกล้รากและหาอนุพันธ์ได้ง่าย

### 4. Secant Method (วิธีเส้นสัมผัส)
- **หลักการ**: คล้ายวิธีนิวตัน แต่ใช้ค่าประมาณอนุพันธ์จากสองจุดแทนอนุพันธ์จริง
- **ข้อดี**: เร็วคล้ายวิธีนิวตัน แต่ไม่ต้องหาอนุพันธ์
- **ข้อเสีย**: ต้องการค่าเริ่มต้น 2 ค่า
- **เหมาะสำหรับ**: กรณีที่หาอนุพันธ์ยาก แต่ต้องการความเร็วสูง

### 5. Cramer's Rule (กฎของเครเมอร์)
- **หลักการ**: แก้ระบบสมการเชิงเส้น AX = B โดยใช้ดีเทอร์มิแนนต์ โดย x_i = det(A_i)/det(A)
- **ข้อดี**: ให้ผลลัพธ์แน่นอน เหมาะสำหรับระบบขนาดเล็ก
- **ข้อเสีย**: ใช้เวลานานมากสำหรับระบบขนาดใหญ่ (O(n!))
- **เหมาะสำหรับ**: ระบบสมการเชิงเส้นขนาด 2x2 หรือ 3x3
